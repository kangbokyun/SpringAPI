# SongLibrary

## 1. 프로젝트 목적
 - 수업 내용 복습과 개념 재확립 및 계층형 게시판 구조 설계와 구조에 맞는 게시글/댓글 등 CRUD 구현
 - 원하는 음악 쉽게 찾기 등의 편의성

## 2. 타겟
 - 노래 듣기를 좋아하는 불특정다수

## 3. 개발환경
 - 운영체제 : Windows10
 - Front-end : HTML, CSS, Bootstrap
 - Back-end : JDK 11, Spring Framework
 - Database : MySQL
 - Server : Tomcat
 - Version Control : Git
